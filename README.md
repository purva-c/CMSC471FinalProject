The Geography of Taste — Michelin Guide Visualization
CMSC 471 Final Project · Spring 2026

CMSC 471 Final Project · Spring 2026 · Sriya Sogal, Sayee Naresh Kumar, Purva Chimurkar
This project visualizes the full Michelin Guide dataset as an interactive scrollable essay, exploring what the geographic distribution of 19,036 restaurant ratings reveals about the guide's history and coverage gaps. It includes a world map, a country-level comparison with a per-capita toggle, a price-versus-stars beeswarm chart, and a section on regions Michelin has never evaluated. All three team members contributed equally to data exploration, design, and implementation. We used AI assistance for portions of the technical work, including map rendering and chart layout, guided by our own design decisions throughout.
